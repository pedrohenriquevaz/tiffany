# Política de Privacidade

**Última atualização: 13/06/2026**

---

## 1. Introdução

Respeitamos sua privacidade. Esta Política de Privacidade explica como este bot Discord (doravante "Serviço") coleta, usa, compartilha e protege suas informações.

---

## 2. Informações que Coletamos

### 2.1 Informações Coletadas Automaticamente

Quando você usa o Serviço, coletamos:

- **ID do Servidor**: Identificador único do servidor Discord
- **ID do Usuário**: Seu identificador único no Discord
- **Mensagens**: Conteúdo das mensagens ao usar comandos do bot
- **Timestamps**: Data e hora das interações
- **Preferências**: Configurações definidas para o bot
- **Uso do Bot**: Quais comandos você executa

### 2.2 Informações Opcionais

Se você fornece:

- **Nome de usuário**: Armazenado para personalization
- **Dados de Configuração**: Preferências personalizadas
- **Feedback**: Se você enviar comentários ao bot

---

## 3. Como Usamos Suas Informações

Usamos as informações coletadas para:

✅ **Funcionalidade do Bot**
- Executar comandos solicitados
- Personalizar sua experiência
- Manter registros de uso (logs)

✅ **Melhorias**
- Analisar uso para melhorar recursos
- Corrigir bugs e problemas
- Desenvolver novas funcionalidades

✅ **Segurança**
- Prevenir abuso do serviço
- Detectar e investigar atividades suspeitas
- Proteger contra spam ou ataques

✅ **Conformidade Legal**
- Cumprir obrigações legais
- Responder a solicitações legais

## 4. Compartilhamento de Dados

### 4.1 Dados NÃO Compartilhados

Nós **NÃO compartilhamos** suas informações com:

- ❌ Terceiros ou serviços de publicidade
- ❌ Outros bots ou aplicativos
- ❌ Plataformas sociais ou redes

### 4.2 Dados que PODEM ser Compartilhados

Suas informações podem ser compartilhadas com:

- **Discord Inc.**: Conforme necessário para funcionamento do bot
- **Provedores de Hospedagem**: Para manter o servidor rodando
- **Autoridades Legais**: Se obrigado por lei

## 7. Exclusão de Dados

### Exclusão Manual

Você pode solicitar a exclusão de seus dados a qualquer momento. Entraremos em contato para confirmar a identidade antes de prosseguir.

### Exclusão Automática

Os dados podem ser automaticamente deletados após **90 dias** de inatividade, conforme política de retenção.

### Exclusão do Servidor

Quando você sair de um servidor ou o bot for removido:
- Os dados associados àquele servidor podem ser retidos por **90 dias**
- Dados pessoais serão marcados para exclusão

Este bot **NÃO usa**:

- ❌ Cookies (não é um website)

Implementamos medidas para proteger seus dados:

- 🔒 Criptografia em trânsito (HTTPS/SSL)
- 🔒 Autenticação segura
- 🔒 Controle de acesso restrito
- 🔒 Logs de auditoria

**Nota**: Nenhum sistema é 100% seguro. Se houver uma violação, notificaremos os afetados.

Este Serviço não é destinado para menores de **13 anos** (conforme Terms do Discord).
